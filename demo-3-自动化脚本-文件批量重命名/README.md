# 批量文件重命名工具 (Batch Rename Tool)

这是一个基于 [Gradio](https://www.gradio.app/) 构建的简单、易用的Web GUI脚本，可以帮助你根据自定义的规则快速批量重命名文件夹中的所有文件。同时它强制引入了 **“先预览效果，后执行”** 的安全机制，防止误操作。

## 🌟 核心功能

*   **添加前缀/后缀**: 快速为文件增加统一前后缀。
*   **文本替换**: 将指定文本替换为新文本（支持正则表达式高级替换）。
*   **统一改名与编号**: 自动重新编号（如 `file_1.jpg`, `file_2.jpg`）。
*   **修改扩展名**: 将某种特定扩展名或所有扩展名批量替换。
*   **全大写/全小写**: 统一转换英文大小写。
*   **操作指引极简**: 只需要复制粘贴你的文件夹路径，即可在浏览器可视化处理。

## 🚀 使用方法

请确保您已安装 Python 和必要的模块 （如 `gradio`）。

1. 打开终端（或在VSCode终端中）。
2. 执行脚本：
   ```bash
   python batch_rename_ui.py
   ```
3. 等待片刻，它会自动在您的默认浏览器中打开页面（通常是 `http://127.0.0.1:7860/`）。
4. **填入待处理的文件夹绝对路径**（例如：`E:\Images`）。
5. 选择重命名规则并输入相应参数。
6. 点击 **"🔍 1. 预览效果"** ，查看表格中的 `新文件名` 是否符合预期。
7. 如果结果满意，点击 **"🚀 2. 确认执行"** 进行真正的重命名。

> **注意：** 重命名操作一旦实际执行，无法自动撤销。因此请务必养成养成“先预览”的好习惯。
